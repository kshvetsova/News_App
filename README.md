# News App
Implemented landing page according to [Figma design](https://www.figma.com/file/d8hE9TLAT6dEV3iCOJnPtS/News-App?node-id=0%3A1) and [API](https://gnews.io/docs/v4#introduction)

- The design 1440px
- Notebook 1280px
- Tablet 768px
- Mobile (> 320px)

Tech stack:

- HTML5, Sass/SCSS
- Flexbox, Grid, BEM
- JavaScript
- React, Hooks, React router
- Fetch, API
